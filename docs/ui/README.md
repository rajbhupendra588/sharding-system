# Sharding System UI

A production-ready web interface for managing the Sharding System. Built with React, TypeScript, and modern web technologies with focus on performance, resilience, and low latency.

## Quick Start

```bash
cd ui
npm install
npm run dev
```

See [docs/ui/README.md](../docs/ui/README.md) for detailed documentation.
