# Configuration Guide

## Overview
This guide explains how to configure the Sharding System.

## Configuration Files
-   `config.yaml`: Main configuration file.
-   `sharding.yaml`: Sharding rules.

## Parameters
### General
-   `port`: The port the server listens on.
-   `log_level`: Logging verbosity (DEBUG, INFO, WARN, ERROR).

### Database
-   `host`: Database host.
-   `port`: Database port.
-   `user`: Database user.
-   `password`: Database password.
