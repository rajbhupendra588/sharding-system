# API Reference

## REST API

### Shards
-   `GET /shards`: List all shards.
-   `POST /shards`: Create a new shard.
-   `GET /shards/{id}`: Get shard details.
-   `DELETE /shards/{id}`: Delete a shard.

### Data
-   `GET /data/{key}`: Get data by key.
-   `PUT /data/{key}`: Put data by key.
